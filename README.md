# yanlingguanwang
研领教育官网

## 部署与访问指南

1. **准备源码**：仓库中的 `index.html`、`assets/css/style.css`、`assets/js/main.js` 和 `assets/images/` 为静态资源，可直接上传至任何静态托管平台。
2. **选择托管**：若需将月成本控制在 ¥3 左右，可使用腾讯云/阿里云对象存储的按量付费模式，或选择 GitHub Pages、Vercel 等免费托管方案。
3. **绑定域名**：在域名提供商处添加 CNAME/A 记录指向托管平台，并开启免费或自动签发的 HTTPS 证书，保证浏览器访问安全。
4. **提交搜索引擎**：部署后将站点地图（如 `sitemap.xml`）及首页链接提交到百度搜索资源平台、360 搜索和必应站长平台，加速收录，让用户能通过浏览器搜索找到官网。

更多上线步骤可参阅首页“上线指南”板块。
